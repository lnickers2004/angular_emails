angular_emails
==============

angular-fullstack demo: angularjs, mongodb, express for CRUD demo

features:

search
pagination
sorting
create
update
delete
list

1. demo url:
http://dixitruth.com:9000/


2. implement locally (make sure your mongod is running):
 
 $ git clone ...;

 $ npm install
 
 $ bower install
 
 $ grunt serve
